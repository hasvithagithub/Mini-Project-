# Smart Traffic Management System 🚦

An AI-based system designed to reduce urban traffic congestion and improve emergency response using deep learning and real-time analysis.

---

## 🚀 Features

* Real-time vehicle detection using YOLOv8
* Emergency vehicle detection using EfficientNet
* Traffic prediction using LSTM
* Dynamic traffic signal control
* Web-based monitoring system

---

##  Technologies Used

* Python
* YOLOv8
* EfficientNet
* LSTM
* OpenCV
* Flask

---

##  How It Works

The system processes live traffic video, detects vehicles, identifies emergency vehicles, and dynamically adjusts signals. It also predicts future traffic using time-series analysis.

---

##  Purpose

To reduce congestion, improve traffic efficiency, and ensure faster emergency response in smart cities.
